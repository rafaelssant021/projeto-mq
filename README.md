# Projeto Media Query (Responsividade)

Acesse o projeto: https://rafaelssant021.github.io/projeto-mq/

## Sobre
Este projeto foi desenvolvido com o objetivo de praticar o uso de Media Queries para criação de layouts responsivos.
A aplicação adapta sua interface de acordo com o tamanho da tela, proporcionando uma melhor experiência tanto em dispositivos móveis quanto em desktops.

---

## Tecnologias utilizadas
- HTML5
- CSS3

---

## Funcionalidades
- Layout adaptável para diferentes tamanhos de tela
- Uso de Media Queries para responsividade
- Ajuste de elementos conforme resolução do dispositivo
- Interface simples e funcional

---


## O que eu pratiquei neste projeto
- Uso de Media Queries para diferentes breakpoints
- Construção de layouts responsivos
- Organização de estilos CSS
- Adaptação de conteúdo para múltiplos dispositivos

---

## Como funciona
O layout utiliza Media Queries para aplicar estilos diferentes conforme a largura da tela, permitindo que o conteúdo se reorganize automaticamente — prática essencial no desenvolvimento web moderno.
